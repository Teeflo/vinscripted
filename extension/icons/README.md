# Icônes de l'Extension

L'extension nécessite 4 fichiers d'icônes au format PNG dans le dossier `extension/icons/` :

- **icon16.png** - 16x16 pixels (toolbar)
- **icon32.png** - 32x32 pixels (toolbar retina)
- **icon48.png** - 48x48 pixels (extension page)
- **icon128.png** - 128x128 pixels (Chrome Web Store)

## Comment créer les icônes

### Option 1 : Générer en ligne
1. Allez sur [Canva](https://www.canva.com) ou [Figma](https://www.figma.com)
2. Créez un design carré avec un symbole "✨" ou "V"
3. Utilisez des couleurs : #667eea (violet) et #764ba2 (pourpre)
4. Exportez en PNG aux 4 tailles

### Option 2 : Utiliser un générateur
- [Icon Generator](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)
- Générez une icône et téléchargez toutes les tailles

### Option 3 : Télécharger des icônes libres
- [Flaticon](https://www.flaticon.com) - Cherchez "sparkles", "magic", "description"
- [Iconfinder](https://www.iconfinder.com)

## Couleurs recommandées

Utilisez les couleurs du gradient de l'extension :
- **Principal** : #667eea (violet bleu)
- **Secondaire** : #764ba2 (pourpre)
- **Fond** : blanc ou transparent

## Format

- **Format** : PNG
- **Fond** : Transparent ou blanc
- **Style** : Simple, flat design

## Exemple minimal

Vous pouvez temporairement utiliser des images vides nommées correctement pour tester, mais vous devrez ajouter de vraies icônes avant de publier sur le Chrome Web Store.
